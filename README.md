# admin-app-backend

Admin app backend with Node JS, Express JS.

Run Project :
`npm start`

Check and Fix es-lint and prettier issues in one command:
`npm format`

Check and Fix issues individually:
Check prettier issues :
`npm format:check`

Fix prettier issues :
`npm format:write`

Check lint issues :
`npm lint:check`

Fix lint issues :
`npm lint:fix`
